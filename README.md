我对于金融市场的波动很感兴趣，计划在传播学等领域建立指标，分析社会思潮的变化

指标设计
- 粉丝量：自媒体粉丝量，计算蓝筹作者、小作者之类的作者指数
	- 播放量：以视频为核心，计算某类 Tag 的指数（后续考虑）
- 数据来源：抖音、哔哩哔哩等交易所数据，打 Tag、排序
- 指数设计：$\sum\limits_{}^{}Followers/Divisor$
- 成分股设计：每年调整一次成分股，比如蓝筹100

指标分析
- 牛市背离：蓝筹很稳、新兴指数暴涨，说明思潮变化加速
- 板块轮动：知识类下跌、玄学类上涨，代表社会开始从外求内
